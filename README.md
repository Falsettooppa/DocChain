# ChainVault

ChainVault is a decentralised document sharing and access control platform built with React, Solidity, web3.js, and Hardhat.

## Features
- Upload documents
- Retrieve documents
- Verify ownership
- Share access using blockchain logic

## Tech Stack
- React
- Solidity
- web3.js
- Hardhat
- Scroll Sepolia

## How it Works
1. User connects wallet
2. User uploads a document
3. Document is registered on-chain
4. Authorised users can retrieve/access the document

## Use Cases
- Academic records
- Legal files
- Business collaboration
- Trusted document exchange
